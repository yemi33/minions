# PR Review Early Bail-Out

Skip full worktree + build + test + lint review cycle when PR commits are unchanged and existing approvals exist.

## Steps

1. **Fetch and check commit SHA**:
   ```bash
   git fetch origin <branch> 2>&1
   git log --oneline -5 origin/<branch>
   ```
   Compare top commit SHA against known prior review state from team notes.

2. **If commit unchanged**, retrieve ADO token:
   ```bash
   TOKEN=$(printf "protocol=https\nhost=office.visualstudio.com\n" | git credential fill 2>/dev/null | grep "^password=" | sed 's/^password=//')
   ```

3. **Post closed-status thread** (status: 4) confirming no action needed:
   ```bash
   cat > "$TEMP/pr-review-thread.json" << 'ENDJSON'
   {"comments":[{"parentCommentId":0,"content":"## Review — No Action Required (Duplicate Dispatch)\n\n**Verdict: APPROVE** ✅\n\nCommit unchanged from prior review. No further action needed.\n\n*Review by Squad (<agent>)*","commentType":1}],"status":4}
   ENDJSON

   curl -s -X POST -H "Authorization: Bearer $TOKEN" -H "Content-Type: application/json" \
     -d @"$TEMP/pr-review-thread.json" \
     "https://dev.azure.com/office/<project>/_apis/git/repositories/<repoId>/pullRequests/<prId>/threads?api-version=7.1"
   ```

4. **Submit APPROVE vote** (vote: 10):
   ```bash
   VSID="1c41d604-e345-64a9-a731-c823f28f9ca8"
   cat > "$TEMP/pr-vote.json" << 'ENDJSON'
   {"vote": 10}
   ENDJSON

   curl -s -X PUT -H "Authorization: Bearer $TOKEN" -H "Content-Type: application/json" \
     -d @"$TEMP/pr-vote.json" \
     "https://dev.azure.com/office/<project>/_apis/git/repositories/<repoId>/pullRequests/<prId>/reviewers/$VSID?api-version=7.1"
   ```

## Notes
- Total time: ~15 seconds vs 5-10 minutes for full review
- If thread count > 30, skip thread posting to avoid ADO rate-limiting
- Vote PUT may time out on high-thread PRs — thread POST is more reliable
- Always use `dev.azure.com` hostname, never `office.visualstudio.com`
