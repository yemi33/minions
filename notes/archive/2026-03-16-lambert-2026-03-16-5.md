### Patterns Discovered

1. **Runtime type guard for CLI string unions in `.devtools/`**: Define `VALID_SCENARIOS` as `readonly ScenarioName[]` and `isValidScenario()` type guard with `includes()` check. Exit with `process.exit(1)` and helpful error listing valid options. Pattern replaces unsafe `as ScenarioName` cast. (source: `.devtools/cowork-demo/src/demo-server.ts:24-33`)

2. **Lazy default value generation in mock providers**: `DEFAULT_OPTIONS` omits `userId` field; it's generated per-call via spread `{ ...DEFAULT_OPTIONS, userId: \`mock-user-\${uuidv4().slice(0,8)}\`, ...options }`. Avoids module-load-time side effects that change behavior per-import. (source: `.devtools/cowork-demo/src/mock-transport/mock-token-provider.ts:38-56`)

3. **`wrappedResolve` pattern for timer leak prevention**: When a Promise has both user-triggered resolution and a timeout fallback, wrap the resolve function to `clearTimeout` the timer on user action. Pattern: `const wrappedResolve = (value) => { if (timeoutTimer) { clearTimeout(timeoutTimer); timeoutTimer = undefined; } originalResolve(value); }`. (source: `.devtools/cowork-demo/src/mock-transport/mock-augloop-server.ts:~415-430`)

4. **Canned fixture validation with monotonic timing**: All fixture sequences (`CannedCoTEvent[]`, `CannedProgressionStep[]`) validate that `delay_ms` values are monotonically non-decreasing in tests. Catches fixture ordering bugs at test time. (source: `.devtools/cowork-demo/tests/fixtures.test.ts:66-72`)

5. **Random port allocation for WebSocket server tests**: `randomPort()` returns `20000 + Math.floor(Math.random() * 20000)` to avoid port conflicts between test suites and parallel runs. (source: `.devtools/cowork-demo/tests/mock-augloop-server.test.ts:18-20`)

6. **`collectMessages` helper for WebSocket testing**: Accepts timeout, collects all parsed messages until close or timeout, resolves Promise with message array. Clean pattern for assertion-based WebSocket testing without callbacks. (source: `.devtools/cowork-demo/tests/mock-augloop-server.test.ts:23-37`)
