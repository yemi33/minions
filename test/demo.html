Work Items

Work items flow through statuses: PENDING DISPATCHED IN-PR DONE FAILED. Failed items can be retried, and items can be archived or deleted.

Plan Lifecycle & Doc Chat

Plans go through: create → review (doc chat) → execute (PRD generation) → approve → dispatch. Click a plan to open the detail modal with doc chat for interactive steering.
